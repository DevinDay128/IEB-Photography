# IEB Photography

A professional wedding photography website for IEB Photography, showcasing elegant design and modern functionality.

## Features

- **Stunning Hero Section** - Full-screen background with elegant typography and call-to-action buttons
- **About Section** - Professional photographer introduction with experience stats
- **Portfolio Gallery** - Filterable image gallery organized by category (Ceremony, Portraits, Details, Reception)
- **Service Packages** - Three-tier pricing structure (Intimate, Classic, Luxe)
- **Testimonials Slider** - Auto-rotating client reviews with touch support
- **Contact Form** - Comprehensive booking inquiry form with validation
- **Fully Responsive** - Optimized for all devices from mobile to desktop

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - pure HTML, CSS, and JavaScript

## Customization

### Replacing Images

Replace the Unsplash placeholder URLs in `index.html` with your own images:
- Hero background: `.hero` section background-image in CSS
- Portfolio images: Update `src` attributes in the portfolio grid
- About section image: Update the image in `.about-image` section

### Contact Information

Update the following in `index.html`:
- Phone number in contact section and footer
- Email address in contact section and footer
- Social media links
